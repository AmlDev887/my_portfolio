О себе

Я Backend-разработчик, специализирующийся на Python и FastAPI. Создаю чистые, хорошо документированные REST API с продуманной архитектурой.

Работаю с реляционными базами данных, строю системы аутентификации, настраиваю Docker-окружения и пишу тесты. Люблю порядок в коде.

Сейчас работаю на практике — активно применяю знания и продолжаю развиваться как специалист.
